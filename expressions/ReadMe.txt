算法思路总结：

需要两个栈 
1、符号栈 2、数字栈

1、符号栈先压一个#符号进栈
2、从表达式中获取一个元素(要么算符要么数字)，并设置为当前元素

while( 当前元素为# 并且 符号栈栈顶元素为# 则 退出循环 )
{
	if 判断当前元素是否为数字
	{	1、压进数字栈中
		2、从表达式中获取下一个元素(要么算符要么数字)，并设置为当前元素
	}
	else
	{
		swith(比较符号栈顶元素和当前元素的优先级关系)
		{
		case >:
			1、符号栈出栈
			2、数字栈出栈 两次
			3、根据运算符合数字进行运算
		case <:
			1、把当前元素压进符号栈
			2、从表达式中获取下一个元素(要么算符要么数字)，并设置为当前元素
		case =:
			1、符号栈出栈
			2、从表达式中获取下一个元素(要么算符要么数字)，并设置为当前元素
		}
	}	

}
